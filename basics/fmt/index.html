Le package fmt implémente des entrées/sorties formatées avec des fonctions analogues aux printf et scanf du language C.

go doc fmt.Print

package fmt // import "fmt"

func Print(a ...interface{}) (n int, err error)
    Print formats using the default formats for its operands and writes to
    standard output. Spaces are added between operands when neither is a string.
    It returns the number of bytes written and any write error encountered.

Afficher le type d’une valeur

fmt.Printf("%T \n", 128)
fmt.Printf("%T \n", "Gopher")

Afficher un booléen

fmt.Printf("%t \n", true)     // true
fmt.Printf("%t \n", false)    // false
fmt.Printf("%t \n", 10 == 10) // true
fmt.Printf("%t \n", 10 > 100) // false

Afficher la valeur d’un entier en base 10

fmt.Printf("%d \n", 255)     // 255
fmt.Printf("%d \n", 128+127) // 255

Afficher la représentation binaire d’un entier

fmt.Printf("%8b \n", 45)  //   101101
fmt.Printf("%08b \n", 45) // 00101101

Afficher les nombres à virgule flottante

fmt.Printf("%f \n", 23.7823)     // 23.7823
fmt.Printf("%.2f \n", math.Pi)   // 3.14
fmt.Printf("%06.f \n", 2450.210) // 002450

Afficher la représentation hexadécimale

fmt.Printf("%x \n", 2500)  // 9c4   valeur sans préfix en minuscule
fmt.Printf("%X \n", 2500)  // 9C4   valeur sans préfix en majuscule
fmt.Printf("%#x \n", 2500) // 0x9c4 valeur avec préfix en minuscule
fmt.Printf("%#X \n", 2500) // 0X9C4 valeur avec préfix en majuscule

Afficher une valeur brute

fmt.Printf("%v \n", "Gopher")  // Gopher
fmt.Printf("%#v \n", "Gopher") // "Gopher"

Afficher une chaînes de caractères entre guillemets

Affiche une chaîne entre guillemets échappée en toute sécurité :

fmt.Printf("%q %q \n", "Hello", "Gopher") // "Hello" "Gopher"

Afficher l’adresse mémoire d’un pointer

var a *int = new(int)

fmt.Printf("%p \n", a) // 0xc000130000

Manipuler l’ordre des verbes

fmt.Printf("Swap: %[2]s %[1]s \n", "Gopher", "Hello") // Swap: Hello Gopher
fmt.Printf("Swap: %[2]d %.2[1]f \n", math.Pi, 360) // Swap: 360 3.14

Scanner une chaîne de caractères

package main

import "fmt"

func main() {
    var name string
    var age uint8

    fmt.Print("Entrez un nom et un age : ")
    fmt.Scanf("%s %d", &name, &age)

    fmt.Printf("Bonjour %s, vous avez %d \n", name, age)
}
