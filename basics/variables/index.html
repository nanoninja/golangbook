Variables

  • Déclarer des variables une par une

  • Opérateur de déclaration de variable court

La déclaration des variables dans Go se fait par l’utilisation du mot-clé var ou l’opérateur de déclaration de variable court :=.

Le mot-clé var peut être utilisé pour déclarer des variables dans un package ou dans une fonction. Il est associé à un nom, un type et une valeur optionnel. Si une variable n’a pas encore reçu de valeur, sa valeur est la valeur zéro de son type.

var NOM TYPE = [EXPRESSION]

L’opérateur de déclaration de variable court := est utilisé pour déclarer et initialiser des variables localement dans des fonctions réduisant ainsi la portée des variables. Le type de la variable est déterminé par le type de l’expression.

NOM := EXPRESSION

Une variable inutilisée, déclarer dans une fonction stoppera la compilation.

Nommage des variables

  • Les noms de variables doivent commencer par une lettre ou un trait de soulignement _ et peuvent contenir les lettres a-z ou A-Z ou les chiffres 0-9 ainsi que le caractère _.
  • Le nom d’une variable est sensible à la casse
  • Les mots-clés ne sont pas autorisés à être utilisés comme nom de variable.
var name = "Gopher"  // valide
var Name = "Gopher"  // valide
var NAME = "Gopher"  // valide
var _NAME = "Gopher" // valide
var 0name = "Gopher" // invalide

Déclarer des variables une par une

Si une variable n’a pas encore reçu de valeur, sa valeur est la valeur zéro de son type (Zero Value).

var name string
var color string
var score int

Déclarer des variables en liste

var (
    name string
    color string
    score int
)

Grouper la declaration des variables par type

var name, color, score string

var (
    name, color string
    score int
)

Déclarer des variables avec un initialisateur

Lorsqu’un initialiseur est présent, le type peut être omis, la variable prendra le type de l’initialiseur et ne pourra plus être changé.

var name = "Gopher"

var (
    color = "Blue"
    score = 300
)

Initialiser des variables en une seule ligne

var name, color, score = "Gopher", "Blue", 300

Opérateur de déclaration de variable court

name := "Gopher"

a, b := 1, 2
b, a := a, b

x, y, z := 0, 0, 0
