Le contrôle de flux se matérialise par l’insertion dans le code d’une structure conditionnelle, (aussi appelé expression conditionnelle). C’est un fonction du langage, qui effectue différents calculs ou actions, en fonction de l’évaluation d’une condition booléenne, à savoir vraie ou fausse.

Les structures conditionnelles dans Go sont similaires à de nombreux langages. Elles sont composées d’un mot-clé réservé, d’un initialisateur optionnel, d’une expression sans parenthèses et d’un bloc d’instructions entre accolades.

if condition ou expression {
    // block d'instruction 
}

for condition ou expression {
    // block d'instruction
}

Structure if / else

L’instruction if évalue la condition :

  • si la condition est évaluée à true, le block d’instruction sera exécuté
  • si la condition est évaluée à false, le block d’instruction ne sera pas exécuté.
package main

import (
    "fmt"
    "os"
)

func main() {
    if len(os.Args) == 2 {
        fmt.Println(os.Args[1])
    }
}

Sortie :

go run main.go hello

hello

Utilisation des branches alternatives de type else et else if :

package main

import (
    "fmt"
    "log"
    "os"
)

func main() {
    if len(os.Args) < 2 {
        log.Fatalln("two missing arguments")
    } else if len(os.Args) == 2 {
        log.Fatalln("the second argument is missing")
    } else {
        fmt.Println(os.Args[1], os.Args[2])
    }
}

Sortie :

go run main.go hello gopher

hello gopher

Switch case simple

Le switch case dans Go est un peu différent du C, le mot-clé break est implicite et les expressions pour chaque cas peuvent-être multiple.

switch [déclaration;] expression {
    case expression1:
        // instructions

    case expression2, expression3, expression4:
        // instructions

    default:
        // instructions
}

package main

import "fmt"

func main() {
    num := 4

    switch num {
    case 2:
        fmt.Println(num)

    case 4:
        fmt.Println(num)
    }
}

Sortie :

4

Switch case avec un initialisateur

package main

import "fmt"

func main() {
    switch value := 4; value {
    case 2:
        fmt.Println(value)

    case 4:
        fmt.Println(value)
    }
}

Switch case conditionnel

package main

import "fmt"

func main() {
    value := "Go"

    switch {
    case len(value) == 1:
        fmt.Println(value)

    case len(value) == 2:
        fmt.Println(value)
    }
}

Switch case multiple

package main

import "fmt"

func main() {
    switch value := 9; value {
    case 2, 4, 6, 8, 10:
        fmt.Println(value)

    case 1, 3, 5, 7, 9:
        fmt.Println(value)
    }
}

Sortie :

9

Switch case de type

package main

import "fmt"

func main() {
    var value interface{}

    value = "Hello Gopher"

    switch t := value.(type) {
    case int, int32, int64:
        fmt.Println(t)

    case string:
        fmt.Println(t)

    default:
        fmt.Println("type is unknown")
    }
}

Switch case avec fallthrough

package main

import "fmt"

func main() {
    value := 75
    switch {
    case value < 50:
        fmt.Println("value is less than 50")
        fallthrough
    case value < 100:
        fmt.Println("value is less than 100")
        fallthrough
    case value < 150:
        fmt.Println("value is less than 150")
    }
}

Sortie :

value is less than 100
value is less than 150

Boucle for

Le langage Go n’utilise que le mot-clé for pour les boucles avec différentes initialisions.

for déclaration; condition; post {
    // block d'instruction à répeter
}
  • la déclaration est exécuté en premier avant la première itération
  • La partie condition est exécutée avant chaque itération.
    Si la condition est fausse, la boucle se terminera sinon la boucle continuera à itérer.
  • La partie post est exécutée après chaque itération. Après cela, la condition est vérifiée,
    si elle est vraie, la boucle se poursuit sinon la boucle existe.
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Print(i)
    }
}

Sortie :

01234

En omettant la condition vous obtiendrez une boucle infinie :

package main

import "fmt"

func main() {
    for i := 0; ; i++ {
        fmt.Print(i)

        if i == 3 {
            break
        }
    }
}

Sortie :

0123

Boucle for ever

La boucle for ever est une boucle infinie. La condition de sortie se fait dans le block d’instruction.

package main

import "fmt"

func main() {
    username := ""

    for {
        fmt.Print("Entrez votre nom: ")
        fmt.Scanf("%s", &username)

        if username != "" {
            break
        }
    }
    fmt.Println("Bonjour", username)
}

Sortie :

go run main.go
Entrez votre nom: 
Entrez votre nom: Gopher
Bonjour Gopher
