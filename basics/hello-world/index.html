Hello World

  • Description du programme

Créer un fichier main.go dans le répertoire de votre choix et ajouter le code suivant :

package main

import "fmt"

func main() {
    fmt.Println("Hello Gophers !")
}

On exécute le programme

go run main.go

Description du programme

Chaque programme doit commencer par la déclaration de package.

Les packages sont utilisés pour organiser et réutiliser le code. Dans Go, un programme est destiné à devenir un exécutable ou une bibliothèque partagée et réutilisable dans d’autres programmes. Dans notre exemple, le package main indique au compilateur qu’il est de type exécutable car c’est le package principale qui contient également la fonction principale point d’entrée de notre application.

package main

Le mot-clé import est utilisé pour importer des packages de type bibliothèque dans votre programme. Le paquet fmt implémente des entrée/sortie formatées avec des fonctions.

import "fmt"

La fonction principale main est le point d’entrée de votre programme. Elle est appelée au moment de l’exécution, ne renvoi rien et ne prend aucun argument.

func main() {
    fmt.Println("Hello Gophers !")
}

Ici, la fonction Println du package fmt est utilisée pour formater le texte vers la sortie standard.

fmt.Println("Hello Gophers !")
