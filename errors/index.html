Gestion des erreurs

  • Renvoyer une erreur depuis une fonction
  • Créer une erreur avec avec le package errors
  • Créer des erreurs réutilisables

Go est un langage qui encourage fortement le traitement des erreurs. Il fournit une interface error qui renvoie un chaîne de caractère de type string. On s’appuie généralement sur le retour multiple des fonctions pour renvoyer une erreur si elle survient. La valeur d’une erreur est une chaîne de caractères correspondant au message, soit à nil qui signifie qu’il n’y a pas d’erreur.

La fonction Errorf du package fmt permet de créer une erreur et de la formater sans passer par le package errors.

package main

import "fmt"

func main() {
    var err error

    fmt.Println(err)

    err = fmt.Errorf("une erreur est survenue")

    fmt.Println(err)
}

Sortie :

<nil>
Une erreur est survenue

Renvoyer une erreur depuis une fonction

Renvoyer une erreur depuis une fonction est le cas le plus courant en Go. Si plusieurs valeurs sont retournée l’erreur sera la dernière dans l’ordre.

package main

import "fmt"

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("division by zero")
    }
    return a / b, nil
}

func main() {
    result, err := divide(10, 0)
    if err != nil {
        fmt.Println(err)
    }
    fmt.Println(result)
}

Sortie :

division by zero
0

Créer une erreur avec avec le package errors

package main

import (
    "errors"
    "log"
)

func main() {
    err := errors.New("mon erreur")

    if err != nil {
        log.Fatal(err)
    }
}

Sortie :

2022/03/05 14:41:37 mon erreur
exit status 1

Créer des erreurs réutilisables

Créer des erreurs personnalisées depuis un package permet de les réutiliser.

package main

import (
    "errors"
    "fmt"
)

var (
    ErrEntityNotFound = errors.New("entity not found")
    ErrInvalidId      = errors.New("invalid ID provided")
)

func find(id string) error {
    if id == "" {
        return ErrInvalidId
    }
    return ErrEntityNotFound
}

func main() {
    err := find("")

    if errors.Is(err, ErrInvalidId) {
        fmt.Println(err)
    }
}

Sortie :

invalid ID provided
