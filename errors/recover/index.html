Panic et Recover

  • Panic et Defer
  • Reprendre le contrôle d’une goroutine paniquée avec recover

Dans la grande majorité du temps la gestion des conditions anormales dans un programme Go se fait en utilisant les erreurs. Elles sont suffisantes pour couvrir l’ensemble d’un programme.

Pourtant il existe certaines situations où un programme après une condition anormale se met à paniquer et se voit contraint d’arrêter prématurément son exécution. Quand une fonction rencontre une panique son exécution est stoppée puis toutes les fonctions différées sont exécutées avant de rendre le contrôle à son appelant.

Les fonctions panic() et recover() sont considérés comme similaires à l’idiome (Try Catch) que l’on peut retrouver dans d’autres langages tels que Java. L’usage des exceptions n’est pas encouragé en Go mais reste néanmoins nécessaire lorsqu’une erreur irrécupérable survient où le programme ne peut simplement pas poursuivre son exécution normalement.

package main

import "fmt"

func doPanic() {
    panic("Panique à bord")
}

func main() {
    doPanic()

    fmt.Println("Fin du programme")
}

Panic et Defer

Toutes les fonctions différées sont exécutées dans l’ordre LIFO avant rendre le contôle à son appelant et terminer le programme.

package main

import "fmt"

func doPanic() {
    panic("Panique à bord")
}

func main() {
    defer func() { fmt.Println("Cleanup 1") }()
    defer func() { fmt.Println("Cleanup 2") }()
    defer func() { fmt.Println("Cleanup 3") }()

    doPanic()

    fmt.Println("Fin du programme")
}

Reprendre le contrôle d’une goroutine paniquée avec recover

La fonction recover() reprend le contrôle d’une goroutine paniquée et n’est utile qu’à l’intérieur des fonctions différées. En opposition à la philosophie de remontée des exceptions dans une classe, il faut que les instructions de récupération soient au plus proche de la source d’erreur.

package main

import "fmt"

func doPanic() {
    defer func() {
        if r := recover(); r != nil {
            fmt.Println(r)
        }
    }()

    panic("Panique à bord")
}

func main() {
    defer func() { fmt.Println("Cleanup") }()

    doPanic()

    fmt.Println("Fin du programme")
}

Sortie :

Panique à bord
Fin du programme
Cleanup
