Pointers

  • Créer un pointeur avec la fonction new

Une variable est une donnée qui doit être accessible instantanément et à tout moment. La mémoire vive est le composant de la machine tout indiqué pour satisfaire ces deux critères.

La mémoire pouvant contenir de nombreuses données, il est nécessaire d’avoir des informations précises pour extraire une valeur. Lorsque l’on recherche une donnée, dans quelle zone mémoire est-elle ? Où commence et s’arrête la lecture d’une valeur ?

Le début de la zone mémoire est donnée par l’adresse et le type de la variable est utilisé pour savoir combien de blocs d’adresse devront être lus. Chaque block de mémoire faire 1 octet soit 8 bits.

var n int32 = 1

    - n correspond à une adresse mémoire
    - n occupe la place de 4 blocks de 8 octets

Données        Adresse mémoire

00000000       0x000001
00000000       0x000002
00000000       0x000003
00000001       0x000004

Lorsqu’une variable est déclarée en tant que pointeur, l’identifiant de la variable ne représente plus le contenu de la variable mais l’adresse mémoire de celle-ci. Pour atteindre le contenu de la variable il faut ajouter le caractère * devant l’identifiant de la variable. À l’inverse pour obtenir l’adresse mémoire d’une variable on ajoute le caractère & devant l’identifiant de celle-ci.

package main

import "fmt"

func main() {
    x := 10 // x a comme contenu 10
    y := &x // y a comme contenu l'adresse mémoire de x

    fmt.Println(*y, y)
}

Sortie :

10 0xc000014098

Créer un pointeur avec la fonction new

Quand une variable est créé sans allocation mémoire le contenu de celle-ci est assignée avec la valeur : nil. Cela signifie que le pointeur n’est pas assigné.

L’assignation d’un pointer peut se faire de façon dynamique via le caractère & ou bien avec la fonction new qui créera et retournera une adresse mémoire après son allocation.

package main

import "fmt"

func main() {
    var x *int32 // on déclare une variable sans l'allouer

    fmt.Println(x) // nil

    x = new(int32) // on alloue la mémoire et on obtient une adresse

    *x = 20 // on modifie la valeur via le pointer pour accéder au contenu

    fmt.Println(*x, x)
}
