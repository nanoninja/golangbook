Un slice est une collection de taille fixe pouvant évoluer soit en effectuant un découpage (slicing) diminuant la taille, soit en utilisant la fonction append() pour ajouter des éléments. Comme le tableau, le slice est composé d’éléments de même type et peut posséder plusieurs dimensions.

var s []type

Déclarer un slice

var s []int

s := []int{}

Initialiser un slice avec la fonction make

La fonction make permet d’initialiser un slice pour un type d’élément donné, en spécifiant une taille et éventuellement sa capacité.

make([]T, length, capacity)

package main

import "fmt"

func main() {
    s := make([]int, 3, 8)

    fmt.Println(s) // [0 0 0]
}

Afficher la taille et la capacité d’un slice

Comme le tableau, la fonction len compte le nombre d’éléments et la fonction cap donne la capacité du slice.

package main

import "fmt"

func main() {
    s := make([]int, 2, 4)

    fmt.Println(len(s), cap(s))
}

Sortie :

2 4

Découper un slice

Il est possible d’obtenir la tranche d’un slice en utilisant l’opérateur [ ] et en spécifiant la position de départ et ou la position de fin. La valeur de départ va de 0 à n-1 et la valeur de fin de 1 à n.

s[start:end]
s[start:]
s[:end]

start:    0       1        2       3       4       5       6       7      8
  end:    1       2        3       4       5       6       7       8      9
slice: ["the", "quick", "brown", "fox", "jumps", "over", "the", "lazy", "dog"]

package main

import "fmt"

func main() {
    s := []string{"the", "quick", "brown", "fox", "jumps", "over", "the", "lazy", "dog"}

    fmt.Println(s[6:9])
}

Sortie:

[the lazy dog]

Ajouter des éléments avec la fonction append

func append(slice []Type, elems ...Type) []Type

package main

import "fmt"

func main() {
    s := []string{"a", "b"}

    s = append(s, "c", "d")

    fmt.Println(s)
}

Sortie :

[a b c d]

Lorsque le nombre d’élément dépasse la capacité initiale d’un slice la fonction append doublera sa taille à partir de la dernière capacité définie. Si vous définissez une capacité de 3 elle passera à 6 puis à 12, 24, etc.

package main

import "fmt"

func main() {
    s := make([]string, 2, 3)

    s[0] = "a"
    s[1] = "b"

    s = append(s, "c", "d")

    fmt.Println("len:", len(s), "cap:", cap(s))
}

Sortie :

len: 4 cap: 6

Fusionnner deux slices avec l’opérateur …

package main

import "fmt"

func main() {
    s1 := []string{"a", "b"}
    s2 := []string{"c", "d"}

    s1 = append(s1, s2...)

    fmt.Println(s1)
}

Modifier une valeur dans un slice

package main

import "fmt"

func main() {
    s := []int{30, 12}

    s[1] = 40

    fmt.Println(s)
}

Sortie :

[30 40]

Supprimer une valeur sans préserver l’ordre

Supprimer une valeur dans un slice sans préserver l’ordre n’est peut-être pas la méthode la plus utilisée mais si l’ordre n’a pas d’importance pour vous cette méthode sera la plus performante.

La suppression se fait en deux étapes :

  • on remplace la valeur à supprimer par la dernière entrée du slice
  • on supprime la dernière entrée du slice en découpant
package main

import "fmt"

func main() {
    s := []string{"a", "b", "c", "d"}

    s[1] = s[len(s)-1]
    s = s[0 : len(s)-1]

    fmt.Println(s)
}

Sortie :

[a d c]

Supprimer une valeur en préservant l’ordre

On tranche deux fois pour récupérer la partie avant et après de la valeur à supprimer. La fonction append prend en premier argument le slice dans lequel on veut ajouter et en deuxième les valeurs à ajouter grâce à la notation … et renvoie un slice contenant toutes les valeurs dans l’ordre.

package main

import "fmt"

func main() {
    s := []string{"a", "b", "c", "d"}

    s = append(s[0:2], s[3:4]...)

    fmt.Println(s)
}

Sortie :

[a b d]

Insérer une valeur au début d’un slice

package main

import "fmt"

func main() {
    s := []string{"b", "c", "d"}

    s = append([]string{"a"}, s...)

    fmt.Println(s)
}

Sortie :

[a b c d]

Insérer une valeur au milieu d’un slice

L’idéal parfois est de découper son l’agorithme en plusieurs petits problèmes plus facile à résoudre. Bien que ça ne soit pas un gros problème à la base ça demande un peu d’habitude.

Dans l’exemple suivant on insére la lettre b entre a et c :

package main

import "fmt"

func main() {
    s := []string{"a", "c", "d"}

    s = append(s[0:1], append([]string{"b"}, s[1:]...)...)

    fmt.Println(s)
}

Sortie :

[a b c d]
