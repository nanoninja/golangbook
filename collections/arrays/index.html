Arrays

  • Déborder la capacité d’un tableau

  • Parcourir un tableau avec range

Un tableau est une collection de taille fixe composé d’éléments de même type et peut avoir plusieurs dimensions. Les éléments sont indicés de 0 à n, L’opérateur [ ] permet lors de la création de définir la taille fixe du tableau et à l’utilisation d’accéder directement à un élément dont l’indice est donnée.

var a [taille]Type

Créer un tableau avec le mot-clé var

package main

import "fmt"

func main() {
    var a [3]string

    fmt.Printf("%#v \n", a)

    a[0] = "a"
    a[1] = "b"
    a[2] = "c"

    fmt.Printf("%#v \n", a)
}

Sortie :

go run main.go

[3]string{"", "", ""} 
[3]string{"a", "b", "c"}

Créér un tableau avec l’opérateur :=

package main

import "fmt"

func main() {
    a := [2]int{34, 56}

    fmt.Println(a)
}

Sortie :

[34 56]

Créer un tableau avec l’anotation …

La notation … spécifie une longueur de tableau égale à l’index d’élément maximum plus un.

package main

import "fmt"

func main() {
    a := [...]string{"x", "y", "z"}

    fmt.Println(a)
}

Déborder la capacité d’un tableau

Si vous débordez la capacité maximale d’un tableau vous obtiendrez une erreur. Go ne permet pas d’ajouter de nouveaux éléments dans un tableau de taille fixe.

# command-line-arguments
./main.go:8:3: invalid array index 2 (out of bounds for 2-element array)

La solution est de créer un nouveau tableau avec une capacité supérieure et de l’alimenter avec les données de l’ancien tableau via une boucle.

package main

import "fmt"

func main() {
    a := [2]string{"x", "y"}
    a2 := [3]string{}

    for i, size := 0, len(a); i < size; i++ {
        a2[i] = a[i]
    }

    a2[2] = "z"

    fmt.Println(a2)
}

Sortie :

[x y z]

Parcourir un tableau

package main

import "fmt"

func main() {
    a := [...]string{"x", "y", "z"}

    for i, l := 0, len(a); i < l; i++  {
        fmt.Println(i, a[i])
    }
}

Sortie :

0 x
1 y
2 z

Parcourir un tableau avec range

package main

import "fmt"

func main() {
    a := [...]string{"x", "y", "z"}

    for i, v := range a {
        fmt.Println(i, v)
    }
}

Sortie :

0 x
1 y
2 z
